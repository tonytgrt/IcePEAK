Documentation

This package has different particle systems ready to be use for all the platforms
Just drag and drop the prefabs in order to use them 
Demo scene is implemented for ease of use, only select the prefab from hieraricy to play the system.
There are different prefabs under "prefabs>>otherprefabs" folder change the rotation of all "x" rotation to -90
If you want to change the number of particles emitting goto the emmition property of the particle system and change the rate over time attribute and min and max brusts accourdingly. 
If you want to change the size of Rain Confetti celebration goto the shape property of the particle system and enlarge their scale accordingly.
To control the drop speed of Rain Confetti change the gravity scale accourdingly.
Initially all the looping on some prefabs are disabled if you want any thing to pop continuously goto each prefabs' particle system and enable the looping attribute
